One page site

Ideal para negócios de pequeno ou médio porte o "one page site" é um site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única pagina.

Wireframe

wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneia o QR Code para obter o wireframe desta página.
